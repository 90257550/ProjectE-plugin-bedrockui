### 基础 EMC 值
```
%projecte_emc%
```
返回玩家的原始 EMC 值

**示例输出:** `1234567`

### 格式化 EMC 值
```
%projecte_emc_formatted%
```
返回带千位分隔符的 EMC 值

**示例输出:** `1,234,567`

### 简短 EMC 值
```
%projecte_emc_short%
```
返回简短格式的 EMC 值，使用 K/M/B/T 后缀

**示例输出:** 
- `1.5K` (1,500)
- `2.3M` (2,300,000)
- `4.7B` (4,700,000,000)
- `1.2T` (1,200,000,000,000)
